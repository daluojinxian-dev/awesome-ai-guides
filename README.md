# 🚀 Awesome AI Guides (AI 进化指南)

[![Deploy Next.js site to Pages](https://github.com/hyapple/awesome-ai-guides/actions/workflows/nextjs.yml/badge.gif)](https://onai.help)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一站式整理全球顶尖 AI 工具、本地大模型搭建教程与提效技巧。助你在这个 AI 时代保持进化。

## 🌟 核心特性

- **高颜值 UI**：基于深色模式、渐变光效和玻璃化卡片的现代设计。
- **极速体验**：使用 Next.js 14 静态导出，秒级搜索与分类过滤。
- **SEO 友好**：针对 AI 关键词深度优化的元数据配置。
- **自动部署**：集成 GitHub Actions，代码提交即刻上线。

## 🛠️ 技术栈

- **框架**: [Next.js 14 (App Router)](https://nextjs.org/)
- **样式**: [Tailwind CSS](https://tailwindcss.com/)
- **动画**: [Framer Motion](https://www.framer.com/motion/)
- **图标**: [Lucide React](https://lucide.dev/)
- **部署**: GitHub Pages

## 📖 快速开始

### 本地开发

1. 克隆仓库:
   ```bash
   git clone https://github.com/hyapple/awesome-ai-guides.git
   ```
2. 安装依赖:
   ```bash
   npm install
   ```
3. 启动开发服务器:
   ```bash
   npm run dev
   ```

### 添加新工具

编辑 `src/lib/data.ts` 文件，在 `AI_TOOLS` 数组中添加新的工具条目即可。

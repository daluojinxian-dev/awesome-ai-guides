# Ollama 入门教程：在本地电脑运行 DeepSeek 与 Llama 3

Ollama 是目前最流行的本地 LLM 运行工具，它将复杂的模型量化和环境配置封装成了极简的命令行工具。

## 为什么选择 Ollama？
* **隐私安全**：数据不出本地，适合处理敏感信息。
* **离线使用**：断网也能正常对话。
* **资源占用低**：支持 4-bit 量化，普通家用电脑也能跑动 7B/14B 模型。

## 快速上手命令
1. **下载模型**：`ollama pull deepseek-v3`
2. **运行模型**：`ollama run deepseek-v3`

## 推荐进阶
配合 **Page Assist** (浏览器插件) 或 **Chatbox**，你可以获得类似 ChatGPT 的优秀 Web 交互体验。

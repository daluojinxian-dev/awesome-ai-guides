# Cursor 彻底改变编程的方式：从 Tab 到全自动工程

Cursor 不仅仅是一个插件，它是一个深度重构的编辑器，旨在让 AI 成为你的结对编程伙伴。

## 杀手级功能：Composer (Ctrl+I)
不再是一个文件一个文件的修改。在 Composer 中输入：“帮我把这个页面的主题换成暗黑模式，并增加一个响应式导航栏”，Cursor 会自动扫描并修改多个相关文件。

## Tab 键的魔力
Cursor 的代码补全不仅是下一行，它能理解你的上下文，直接补全整个代码块甚至复杂的逻辑判断。

## 小贴士
* **导入配置**：由于基于 VS Code，你可以一键导入你所有的插件和快捷键。
* **模型选择**：在设置中开启 Claude 3.5 Sonnet，你会发现它的逻辑理解力远超 GPT-4o。

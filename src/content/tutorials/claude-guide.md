# Claude 3.5 Sonnet 使用指南：不仅是地表最强编程助手

Claude 3.5 Sonnet 是 Anthropic 推出的划时代模型，它在代码编写、视觉理解和幽默感上都达到了行业顶尖水平。

## 核心亮点：Artifacts 窗口

这是目前 AI 交互的最佳创新。当你请求代码、网站原型或矢量图时，Claude 会在右侧弹出一个实时预览窗口。

### 如何高效利用 Artifacts？
1. **即时渲染**：让它写一个 React 组件，你可以直接在右侧看到运行效果。
2. **代码迭代**：直接在对话框要求：“把紫色换成深蓝色”，预览窗口会无缝更新。
